# Gestion Logistique Camions

> Application de gestion des entrées camions, pointage et suivi logistique

## Description

Cette application Android/Java permet de gérer les opérations logistiques liées aux camions :
- Enregistrement des entrées et sorties de camions
- Pointage des chauffeurs et véhicules
- Suivi en temps réel des mouvements logistiques
- Génération de rapports d'activité

## Technologies

- **Plateforme** : Android
- **Langage** : Java
- **Base de données** : SQLite / Room

## Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/wanisius/gestion-logistique-camions.git
   ```
2. Ouvrir le projet dans Android Studio
3. Synchroniser les dépendances Gradle
4. Lancer l'application sur un émulateur ou appareil physique

- [ ] Suivi logistique en temps réel
