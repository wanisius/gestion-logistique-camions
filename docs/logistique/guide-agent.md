# Guide Agent — Application Gestion Logistique

## Introduction

Ce guide explique comment utiliser l'application mobile pour la gestion quotidienne des camions.

## Démarrage

1. Ouvrir l'application **Gestion Logistique Camions**
2. Se connecter avec vos identifiants
3. Vérifier que la date/heure est correcte

## Opérations courantes

### Enregistrer un camion entrant

1. Appuyer sur **"+ Nouveau camion"**
2. Saisir la plaque d'immatriculation
3. Saisir le nom du chauffeur
4. Sélectionner le type de marchandise
5. Ajouter des observations si nécessaire
6. Appuyer sur **"Enregistrer"**

### Effectuer un pointage de sortie

1. Retrouver le camion dans la liste
2. Appuyer sur le camion concerné
3. Appuyer sur **"Pointer la sortie"**
4. Confirmer l'action

### Consulter le tableau de bord

- Voir le nombre de camions sur site
- Consulter les entrées du jour
- Vérifier les camions en attente

## Problèmes fréquents

| Problème | Solution |
|---|---|
| Application lente | Redémarrer l'application |
| Données non sauvegardées | Vérifier l'espace de stockage |
| Erreur de connexion | Contacter le support technique |

## Support

En cas de problème, contacter le responsable logistique ou le service informatique.
